Hybrid RANS-LES simulations of the CRM Wing Body Tail configuration. Each file contains results for a family of 3 mesh resolutions: R1.0 (baseline), R1.4 (refinement factor sqrt(2) in each coordinate direction), and R2.0 (refinement factor 2 in each coordinate direction). All simulations performed using a time-step of 1000 steps per CTU (MAC/Umag_freestream). Non buffet AOAs were run for 200 CTUs with averaging starting after 80 CTUs. Buffet cases were performed much longer (approx. 400-500 CTUs) again with averaging starting after 80 CTUs).
